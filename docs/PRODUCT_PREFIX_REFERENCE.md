# Product Prefix Reference Guide

This document lists all product display prefixes used in the system for POS loading slips and quotations.

## Product Prefixes by Category

### Base Plates
- **BP** - Base Plate

### Sheets
- **SH** - CR Sheet (Cold Rolled)
- **SH** - GI Sheet (Galvanized Iron)
- **SH** - HR Sheet (Hot Rolled)
- **ROOF** - Roofing Sheet

### Cut Plates
- **CP** - Cut Plate

### MS Structurals
- **ANG** - MS Angle
- **CH** - MS Channel
- **MSIB** - MS I Beam

### MS Flats
- **FL** - MS Flat

### MS Pipes
- **P-CHS** - MS Pipe CHS (Circular Hollow Section)
- **P-RHS** - MS Pipe RHS (Rectangular Hollow Section)
- **P-SHS** - MS Pipe SHS (Square Hollow Section)

### MS Round and Square
- **RD** - MS Round
- **SQ** - MS Square

### MS Bright Bar
- **BR** - MS Bright Bar (Round)
- **BR-FL** - MS Bright Bar Flat
- **BR-SQ** - MS Bright Bar Square
- **BR-HEX** - MS Bright Bar Hexagon

### TMT Rods
- **TMT** - TMT Rod

### Other Categories
- **GEN** - Generic Product
- **PNF** - Product Not Found
- **UNK** - Unknown (fallback for uncategorized items)
